# Portofolio Irenne (Pink Theme)
Cara pakai:
1. Copy folder `portofolio-irenne` ke `C:\laragon\www\`.
2. Jalankan Laragon (Start All).
3. Buka `http://localhost/portofolio-irenne`.

Opsional:
- Untuk form kontak nyata: pasang PHPMailer dan konfigurasi SMTP.
- Untuk menambah halaman: duplicate header/nav dan sesuaikan.
